# Ecshop56
> Build in 2017 / 02 / 23



### Ecshop支持php5.6版本（包括5.4\5.5\5.6）


### 安装非常简单
> 无需与原本一样，执行安装过程
> 
> 参考文档： ```ecshop-支持PHP5.6-安装说明2017-02-23.doc```

+ 将项目直接拷贝你的 web 项目存放目录下 
+ 导入数据库文件 ```/data/ecshop_php56.sql```
+ 后台账号密码 
	
	+ ```admin```  
	+ ```admin123```

+ 修改数据库配置文件  ```data/config.php```
